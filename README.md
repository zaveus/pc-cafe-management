# PC Café POS System  
*A lightweight POS and time-tracking app for modern PC cafés.*

## Overview
This project is a simplified Point-of-Sale and user-session management system built for PC café environments.  
It was developed as part of a database course project and showcases practical full-stack skills using React (Vite + TypeScript), PHP, and MySQL.

## Features
- **User Login & Session Timer**  
  Tracks remaining time and updates the UI in real-time.
- **Membership Card / ID Verification**  
  Basic ID check to validate user access.
- **Store Module**  
  Allows users to “order” items such as snacks, drinks, and peripherals.
- **Admin Dashboards**  
  - **User Dashboard:** manage accounts and session credits  
  - **Store Dashboard:** add/edit store items  
  - **Order Queue:** view active and completed orders  
- **Secure Database Integration**  
  Clean MySQL schema for users, sessions, orders, and inventory.

## Technologies Used
- **Frontend:** React, Vite, TypeScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Styling:** CSS (games.css, store.css)


## Notes
- This is an **educational project** — architecture is intentionally simple.  
- Future improvements may include: real payment integration, advanced auth, membership cards with QR/RFID, and optimized backend logic.
